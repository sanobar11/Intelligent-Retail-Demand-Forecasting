# Intelligent Retail Demand Forecasting System

## Project Overview
This project predicts retail product demand using time series forecasting techniques.

## Models Used
- ARIMA (Statistical Time Series Model)
- LSTM (Deep Learning Model)

## Technologies
Python, Pandas, NumPy, TensorFlow, Matplotlib, Scikit-learn

## Features
- Data preprocessing
- Time series analysis
- Model comparison
- Demand prediction visualization

## Applications
Retail inventory planning and demand optimization.
